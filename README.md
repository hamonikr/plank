# Install

## HamoniKR (>= 3.0)
```
sudo apt update
sudo apt install -y plank
```
## Ubuntu (>= 18.04), LinuxMint (>= 19)
```
wget -qO- https://pkg.hamonikr.org/add-hamonikr.apt | sudo -E bash -
sudo apt plank
```
